# StudyBase

O **StudyBase** é uma base pessoal de conhecimento para organizar qualquer tipo de aprendizado de forma simples, estruturada e fácil de consultar.

A proposta do projeto é permitir que o usuário salve anotações, explicações, comparações, exemplos práticos, códigos, erros comuns e soluções em um único lugar, com busca rápida por palavra-chave, áreas, categorias e tags.

## Objetivo

Criar uma aplicação web onde o usuário possa registrar e consultar seus próprios aprendizados de maneira organizada.

O sistema poderá ser usado para qualquer assunto, como:

* programação;
* inglês;
* faculdade;
* musculação;
* finanças;
* livros;
* projetos pessoais;
* estudos em geral.

## Problema que o projeto resolve

Durante os estudos, é comum aprender algo importante, resolver um erro ou entender uma diferença entre conceitos, mas depois ter dificuldade para encontrar essa informação novamente.

O **StudyBase** resolve essa dor criando uma base pessoal onde o usuário pode salvar o conhecimento de forma estruturada e encontrar rapidamente quando precisar.

## Diferencial

O StudyBase não será apenas um app genérico de notas.

A ideia é permitir que o usuário organize o conhecimento por contexto, usando campos como:

* área;
* categoria;
* tags;
* tipo de anotação;
* explicação;
* exemplo prático;
* quando usar;
* comparação;
* erros comuns;
* solução.

## Tipos de anotação

O sistema poderá trabalhar com diferentes tipos de anotação:

### Anotação simples

Para conceitos rápidos, frases, regras, lembretes e observações.

### Guia

Para explicações mais completas, passo a passo e conteúdos de estudo.

### Comparação

Para comparar duas ou mais opções.

Exemplos:

* fetch vs axios;
* supino reto vs supino inclinado;
* SQL vs NoSQL;
* present perfect vs simple past.

### Snippet

Para salvar códigos, comandos ou trechos reutilizáveis.

### Erro e solução

Para registrar problemas encontrados, causa provável e solução aplicada.
